# NestJS Starter Pack

Complete NestJS AI tooling pack — installs rules, a module scaffold command, and a PR review workflow in one go.

## Included

| Asset                               | Type     | Description                                    |
| ----------------------------------- | -------- | ---------------------------------------------- |
| `lozymon/nestjs-rules`              | rule     | NestJS coding conventions for AI tools         |
| `lozymon/nestjs-generate-module`    | command  | Slash command to scaffold a feature module     |
| `lozymon/nestjs-pr-review-workflow` | workflow | PR review checklist for NestJS rule violations |

## Install

```bash
npx ruleshub install lozymon/nestjs-starter-pack --tool claude-code
```
